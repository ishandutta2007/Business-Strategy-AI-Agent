# Business-Strategy-AI-Agent
## Top Business Strategy Automation Tools Ecosystem

**Curated List of SaaS Products & Open-Source GitHub Projects**  
*Focused on AI-Powered Strategy Planning, Execution & Business Intelligence Automation*  
**Last updated: March 2026**

This repository tracks notable **SaaS platforms** and **open-source projects** building **Business Strategy Automation Tools**. These tools help leadership teams automate strategic planning, OKR tracking, scenario modeling, initiative execution, performance monitoring, and decision intelligence using AI agents and workflows.

**Examples** include Foray AI, CoachOS, n8n, Lindy, Google NotebookLM, WorkBoardAI, OnePlan, Cascade Strategy, Precision.co, ThoughtSpot, and Gumloop (the category leaders). Tools listed here emphasize **agentic capabilities** (multi-step reasoning, automated reporting, adaptive strategy adjustment, and cross-team orchestration).

**Open-source emphasis**: This section is heavily expanded with every major active project for self-hosting, local LLMs (Ollama), full data privacy, and complete customization — ideal for organizations and teams who want sovereign strategy automation without vendor lock-in.

Contributions welcome! Open a PR to add/update entries. Keep descriptions factual and link to official sites.
